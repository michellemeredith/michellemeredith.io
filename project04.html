Guessing Game

Languages & Software: Rust

This is a number guessing game written in the systems programming language Rust. I started to learn Rust after being intrigued by its guarantee of thread safety without requiring garbage collection.

The program has the user guess a number and continues until the user guesses correctly, or until the user enters 0 to quit.
