Password Generator

Languages & Software: Java; IntelliJ Idea

This is a program that generates randomized passwords.

There are two main options for a new password:
- Default: a password containing 4 words chosen at random.
- Customized: the user can choose how many words are used, how many words are capitalized, and how many symbols are used in the password.

Examples:
"d": generates a default password consisting of four words chosen at random.
"w 3": three random words will be used.
"n 2": two random numbers (0-9) will be used.
"c 2": two words will be capitalized.
"s 1": one symbol will be included.

I'm planning on adding the implementing the following feautures in future iterations of this project:
- The ability to set a range on word length (ex: 4-6 letter words), which would make the password easier to commit to memory.
- Pulling the words from a dictionary API that supports different languages so that the user can generate a password in the language of their choosing.
